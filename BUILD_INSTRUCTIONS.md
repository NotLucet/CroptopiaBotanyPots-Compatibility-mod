# Инструкции по сборке

## Требования

- **Java Development Kit (JDK) 21**
- **Git** (опционально)

## Сборка

```bash
# Windows
.\gradlew.bat build

# Linux / macOS
./gradlew build
```

## Где найти JAR

```
build/libs/croptopiabotany-1.0.0.jar
```

## Режим разработки

```bash
# Запуск клиента
.\gradlew.bat runClient

# Очистка
.\gradlew.bat clean
```
