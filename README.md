# Croptopia Botany Pots Compat

[![Minecraft](https://img.shields.io/badge/Minecraft-1.21.1-green.svg)](https://minecraft.net/)
[![NeoForge](https://img.shields.io/badge/NeoForge-21.1.x-red.svg)](https://neoforged.net/)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Мод добавляет полную совместимость между **Croptopia** и **Botany Pots** для Minecraft 1.21.1 на NeoForge.

## Особенности

- **85 рецептов** для выращивания всех растений Croptopia в горшках Botany Pots
- Все овощи, зелень и ягоды (57 видов)
- Все фруктовые деревья (28 видов)
- Правильные ID предметов и блоков
- Сбалансированное время роста (60-70 секунд)
- Корректный дроп с шансом на повторное получение саженца

## Требования

| Мод | Версия | Обязательно |
|-----|--------|-------------|
| **Minecraft** | 1.21.1 | Да |
| **NeoForge** | 21.1.x | Да |
| **Croptopia** | 4.2.x | Да |
| **Botany Pots** | 21.1.x | Да |

## Установка

1. Установите **Minecraft 1.21.1** с **NeoForge 21.1.x**
2. Установите моды **Croptopia** и **Botany Pots**
3. Скачайте `croptopiabotany-1.0.0.jar`
4. Поместите файл в папку `mods/`
5. Запустите игру

## Использование

1. Поставьте горшок Botany Pots
2. Нажмите **ПКМ** для открытия интерфейса
3. Положите **семя или саженец Croptopia** в слот
   - Можно использовать **Shift+ЛКМ** для быстрой вставки
4. Убедитесь, что в горшке есть подходящая почва
5. Дождитесь роста (60-70 секунд)
6. Соберите урожай!

## Сборка из исходников

### Требования
- **Java 21** (JDK)

```bash
./gradlew build
```

Скомпилированный JAR: `build/libs/croptopiabotany-1.0.0.jar`
